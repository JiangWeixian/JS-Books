# 用户界面
> 试图和系统一致

## 系统颜色字体等

CSS支持很多关键字，这些关键字具体内容的一般都是系统关键字，可以和系统某部分内容样式一致。

可以通过`font`设置系统字体，还有`color`的系统颜色。

## cursor

可以设置`url`来进行图像指针，如果设置为`help`，会获取哪个元素的`title`信息。
